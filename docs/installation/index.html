From source code¶

These instructions have been tested for Anaconda python. You can use any python distribution you like at risk of more complications.

  1. Clone the source code from https://github.com/glebdovzhenko/P61Viewer . You can install git for free and keep the

project up to date, or just download the source code form github and unpack the archive.

anyfolder $ git clone https://github.com/glebdovzhenko/P61Viewer.git

  1. Set up a virtual environment (python version should be 3.5 since pyinstaller does not support newer versions yet)

anyfolder $ conda create -n P61Venv python=3.5
anyfolder $ conda activate

  1. Change the directory to the project root

anyfolder $ cd P61Viewer

  1. Update pip and install the dependencies

P61Viewer $ pip install pip --upgrade
P61Viewer $ pip install -r requirements/base.txt

4.a If you are using Mac OS X, install the framework version of python and make sure (step 5.a) that it is the interpreter you will be using to launch the application. It is necessary for most GUI python applications on Mac

P61Viewer $ conda install -c anaconda python.app

  1. (Windows, Linux) Change the directory to src and launch the application

P61Viewer $ cd src
src $ python P61ViewerMain.py

5.a (Mac OS X) Change the directory to src and launch the application

P61Viewer $ cd src
src $ pythonw P61ViewerMain.py
