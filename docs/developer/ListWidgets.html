ListWidgets¶

Interface¶

This sub-module interfaces two classes: EditableListWidget and ActiveListWidget.

  • EditableListWidget represents a list of datasets. Allows for group operations such as adding, deleting, changing active status.

  • ActiveListWidget represents a list of active datasets. Allows for having selected just one item at a time.
